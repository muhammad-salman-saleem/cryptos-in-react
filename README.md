# React Material UI Dashboard

## Getting started

### How to run frontend 
```
npm install
npm run start
```

### How to run backend 
```
cd api
npm install
npm run start
```

## Technology stack

- [Material UI V5](https://mui.com/core/)
- Built with React Hooks API.
- Redux & React context API for state management.
- Redux toolkit.
- React Router for navigation routing.
- Support for react-script.
- Code splitting.
- CSS-in-JS.
